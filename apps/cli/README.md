# CLI - AnimeList

The purpose of creating this CLI is to make it easier and faster to work with the anime list stored in the .yaml file.

As the list grows, it will become increasingly difficult to manage, search, relate entries, or add new ones manually.

A graphical interface isn’t planned, as the goal is to keep things simple and lightweight.

Instead, a CLI tool will provide quick access and management features.

Ultimately, this project is intended to be self-hosted, so anyone can fork it and manage their own anime list independently.
